## Reportando uma vulnerabilidade

**Não abra issues públicas para vulnerabilidades.**

Envie um e-mail para **joaopaulomirandamatias@gmail.com** com:

- Descrição da falha e impacto esperado.
- Passos para reproduzir (PoC se possível).
- Versão afetada (commit hash ajuda).

Prazo de resposta inicial: até 7 dias corridos. Após validação, uma correção
será priorizada e divulgada via *GitHub Security Advisory*.

Obrigado por ajudar a manter a comunidade segura.
